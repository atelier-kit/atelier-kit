# Atelier-Kit adapter: Antigravity

Use the adapter instruction file generated by `atelier install-adapter antigravity`.

- Planning stays Antigravity-native until someone explicitly runs `/atelier ...`.
- `/atelier quick|plan|deep ...` activates Atelier through the `atelier` CLI.
- When active, read `.atelier/active.json`, the active epic `state.json`, and only `.atelier/skills/<active_skill>.md`.
- At `planned`, implement with Antigravity's native workflow from the exported plan, then run `atelier review`.
